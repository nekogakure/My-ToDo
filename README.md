# My ToDo
これはねこがくれが作成した、簡単かつ軽量なToDoソフトです。簡易的ながらも、LINEと連携させることができ、通知を送信することもできます。

## 使い方
1. LineNotifyのトークンを発行する。（自分で調べてください）
2. トークンをソースコードに張り付ける
3. サーバーにアップロードする

たったこれだけで、だれでもToDoアプリを使用することができます。また、これだと誰でもアクセスできてしまうのでBasic認証をかけることで擬似的にアカウントを作れます。

## 機能
- タスクの追加
- タスクの削除
- タスクにコメントをつける
- タスクの完了、未完了が一目でわかる
